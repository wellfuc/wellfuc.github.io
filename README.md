# Wellfuc's GitHub Page

Welcome to my personal GitHub Pages site! This site is built using Jekyll and hosted on GitHub Pages.

## About

This is a simple Jekyll-based website showcasing my projects, thoughts, and more. It's automatically built and deployed from the `main` branch.

## Local Development

To run this site locally:

1. Install Jekyll and Bundler:
   ```bash
   gem install jekyll bundler
   ```

2. Clone the repository:
   ```bash
   git clone https://github.com/wellfuc/wellfuc.github.io.git
   cd wellfuc.github.io
   ```

3. Install dependencies:
   ```bash
   bundle install
   ```

4. Run the site:
   ```bash
   bundle exec jekyll serve
   ```

5. Open your browser to `http://localhost:4000`

## Customization

- Edit `index.md` for the home page content.
- Modify `_config.yml` for site settings.
- Update styles in `assets/css/style.css`.
- Add new pages in the root directory with front matter.

## Deployment

The site is automatically deployed via GitHub Pages when changes are pushed to the `main` branch.

For more information, visit [GitHub Pages documentation](https://docs.github.com/en/pages).
